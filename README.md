# Fancify My Text 🎨

A fun, interactive web app that lets you style and transform text right in your browser — no installs, no frameworks, just HTML, CSS, and vanilla JavaScript.

**Live site:** [hashtags2023.github.io](https://hashtags2023.github.io)

---

## Features

- **Text input** — Type or paste any text into the input area
- **Bigger!** — Increases the font size of your text
- **FancyShmancy / BoringBetty** — Toggle between a decorative style and a plain style
- **Moo** — Applies a special "Moo" effect to your text

## Getting Started

No build step required. Just open `index.html` in your browser:

```bash
git clone https://github.com/hashtags2023/hashtags.github.io.git
cd hashtags.github.io
open index.html
```

Or visit the live GitHub Pages site directly: [https://hashtags2023.github.io](https://hashtags2023.github.io)
